# Online Magazine Website
This is a responsive online magazine website.

## Features :
- Home
- Categories :
  - Sports
  - Health
  - Life
  - Politics
  - Business
  - Technology
- Article title with username
- Recent / Popular
- More articles by the author
- Post Comments
