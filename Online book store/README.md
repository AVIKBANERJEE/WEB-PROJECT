# Online Book Store
This is a responsive Online Book Store website.

## Features:
- Search Bar
- Wishlist
- Add to cart
- User Profile
  - Sign in page
  - Remember Password
  - Forget Password
  - Sign up option [If don't have any account]
- Home
- Featured books
- New arrival
- Reviews
- Blogs
- Contact info
- Extra links
- Locations
- Social media handles
