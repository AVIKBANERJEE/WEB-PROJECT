# Responsive Admin Website
This is a fully responsive admin dashboard design.

## Features:
- At a glance view of
  - Daily views
  - sales
  - comments
  - Earnings
- Recent Customers
  - Name
  - Order Price
  - Payment status

- Side Navbar
  - Dashboard
  - Customer
  - Message
  - Help
  - Setting
  - Password
  - Sign out
